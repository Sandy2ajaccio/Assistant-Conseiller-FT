# Règles métier – Projet professionnel

## Objectif

Définir les règles permettant d'évaluer la maturité d'un projet professionnel et de proposer les actions les plus adaptées.

---

# FT-PROJET-001

## Intitulé

Projet professionnel insuffisamment défini

### Objectif

Identifier les situations dans lesquelles le projet n'est pas suffisamment construit pour engager directement une formation ou une recherche ciblée.

### Critères

Le projet est considéré comme insuffisamment défini lorsque l'un ou plusieurs des éléments suivants sont observés :

- le métier recherché n'est pas clairement identifié ;
- plusieurs projets sont évoqués sans priorité ;
- le demandeur ne connaît pas le métier visé ;
- le projet change fréquemment ;
- aucun argument ne justifie le choix du métier.

### Analyse métier

La priorité est de clarifier le projet avant toute prescription importante.

### Actions envisageables

- approfondir l'entretien ;
- explorer les compétences et centres d'intérêt ;
- proposer une prestation d'aide à la définition du projet lorsque cela est pertinent ;
- envisager une PMSMP si une immersion peut aider à confirmer un métier.

### À éviter

- prescrire immédiatement une formation qualifiante ;
- construire une MAP uniquement fondée sur un projet non stabilisé.

### Résultat attendu

Le demandeur est capable d'exprimer un projet professionnel cohérent, argumenté et réaliste.

# FT-PROJET-002

## Intitulé

Projet professionnel mature

### Critères

- métier clairement identifié ;
- motivation stable ;
- bonne connaissance du métier ;
- cohérence avec les compétences ou un parcours réaliste ;
- contraintes connues.

### Analyse métier

Le projet peut désormais être consolidé par une recherche active d'emploi, une immersion, une formation ou une mise en relation.

### Actions envisageables

- recherche d'offres ciblées ;
- PMSMP ;
- formation adaptée si un écart de compétences est identifié ;
- mise en relation avec un partenaire si nécessaire.

### Résultat attendu

Le demandeur dispose d'un parcours d'accès au métier clairement défini.

---

# FT-PROJET-003

## Intitulé

Projet nécessitant une validation terrain

### Critères

Le projet paraît cohérent mais n'a jamais été confronté à la réalité du métier.

### Analyse métier

Une immersion professionnelle peut permettre de confirmer ou d'infirmer le projet.

### Actions envisageables

- rechercher une entreprise d'accueil ;
- définir les objectifs de la PMSMP ;
- préparer un bilan à l'issue de l'immersion.

### Résultat attendu

Le projet est confirmé, ajusté ou abandonné sur la base d'une expérience concrète.
