# Dossier Demandeur

## Objectif

Le dossier demandeur est le cœur du logiciel.

Tous les autres modules s'appuient sur ce dossier.

Aucune information ne doit être dupliquée.

---

# Informations générales

- Identifiant interne
- Nom
- Prénom
- Date de naissance
- Âge
- Sexe
- Adresse
- Commune
- Téléphone
- Mail

# Historique

Le logiciel conserve :

- tous les entretiens
- toutes les actions
- toutes les prescriptions
- toutes les synthèses
- tous les courriers
- toutes les décisions

Aucune donnée ne doit être supprimée.

---

# Intelligence artificielle

L'IA pourra :

- analyser le dossier
- détecter les incohérences
- proposer des actions
- préparer un entretien
- générer une synthèse
- suggérer des dispositifs

L'IA ne décide jamais à la place du conseiller.
