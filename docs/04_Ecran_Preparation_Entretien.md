# Écran : Préparation d'entretien

## Objectif

Préparer un entretien en moins de 2 minutes.

---

# Résumé IA

Afficher automatiquement :

- situation actuelle
- évolution depuis le dernier entretien
- principaux changements
- points positifs
- points de vigilance

---

# Vérifications

Le conseiller vérifie :

- coordonnées
- disponibilité
- projet professionnel
- recherche d'emploi
- formations
- actions réalisées

---

# Questions proposées

Le moteur IA propose :

- Questions sur le projet
- Questions sur les freins
- Questions sur les démarches
- Questions sur la motivation

---

# Contrôles automatiques

Détecter :

- contrat d'engagement absent
- entretien en retard
- dossier incomplet
- document manquant
- échéance proche

---

# Recommandations

Le moteur propose :

- prestation
- formation
- PMSMP
- immersion
- IAE
- orientation partenaire
- aide financière

---

# Documents utiles

Afficher automatiquement :

- CV
- dernier compte-rendu
- contrat d'engagement
- prescriptions
- courriers

---

# Boutons

- Démarrer l'entretien
- Modifier le dossier
- Ajouter une action
- Ajouter une note
- Générer une synthèse

---

# Contraintes

Le conseiller doit pouvoir préparer un entretien complet en moins de deux minutes.
