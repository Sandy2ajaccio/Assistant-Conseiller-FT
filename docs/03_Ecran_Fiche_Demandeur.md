# Écran : Fiche Demandeur

## Objectif

Permettre au conseiller d'avoir une vision complète du demandeur en un seul écran.

---

# En-tête

Afficher :

- Nom
- Prénom
- Âge
- Numéro identifiant
- Conseiller référent
- Date d'inscription
- Catégorie
- Dispositif
- Statut

---

# Résumé

Afficher sous forme de cartes :

- RSA
- RQTH
- CEJ
- IAE
- Formation
- Contrat d'engagement
- Disponible
- Dernier entretien

---

# Freins

Cases colorées :

- Santé
- Mobilité
- Logement
- Garde d'enfant
- Numérique
- Langue
- Justice
- Endettement

---

# Projet professionnel

Afficher :

- Métier recherché
- Niveau
- Diplômes
- Compétences
- Expériences
- Formations réalisées

---

# Historique

Timeline :

- Entretiens
- Notes
- Actions
- Formations
- Prescriptions
- Courriers
- Orientations

---

# Plan d'action

Afficher :

- Actions en cours
- Actions terminées
- Actions en retard

---

# Documents

Liste :

- CV
- Contrat d'engagement
- CER
- Justificatifs
- Courriers

---

# Assistant Expert

Toujours visible à droite.

Il affiche :

- Résumé automatique
- Priorités
- Alertes
- Recommandations
- Questions à poser
- Prochaine action conseillée

---

# Boutons

- Modifier
- Ajouter une note
- Ajouter une action
- Préparer l'entretien
- Générer une synthèse IA
- Historique complet

---

# Contraintes

Aucune fenêtre inutile.

Toutes les informations importantes visibles immédiatement.

Le conseiller ne doit presque jamais changer d'écran.
