# Écran : Tableau de bord

## Objectif

Permettre au conseiller de voir immédiatement les informations prioritaires dès son arrivée.

---

# En-tête

- Nom du conseiller
- Date
- Heure
- Barre de recherche
- Notifications

---

# Bloc 1 : Aujourd'hui

- Rendez-vous du jour
- Entretiens
- Ateliers
- Échéances

---

# Bloc 2 : Mes priorités

- Contrats d'engagement à renouveler
- PASS IAE arrivant à échéance
- Dossiers sans entretien
- Dossiers sans action
- Alertes

---

# Bloc 3 : Mon portefeuille

Afficher :

- Nombre total
- Nouveaux inscrits
- RSA
- Jeunes
- Seniors
- RQTH
- IAE

---

# Bloc 4 : Assistant Expert

Le moteur expert affiche :

- Priorité n°1
- Priorité n°2
- Priorité n°3

Puis :

- anomalies détectées
- opportunités
- recommandations

---

# Bloc 5 : Raccourcis

Boutons :

- Nouveau dossier
- Recherche
- Convocation
- Courrier
- Analyse IA
- Statistiques

---

# Contraintes

Le tableau de bord doit être utilisable sans formation.

Le conseiller doit comprendre l'écran en moins de 10 secondes.

Aucune information inutile.
