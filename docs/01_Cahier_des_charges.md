# Cahier des charges fonctionnel
## Assistant Conseiller FT

Version : 1.0

---

# Vision du projet

Assistant Conseiller FT est un logiciel d'aide à la décision destiné aux conseillers France Travail.

Son objectif est de :

- faciliter le suivi des demandeurs d'emploi ;
- sécuriser les décisions ;
- automatiser les tâches répétitives ;
- centraliser les informations ;
- assister le conseiller grâce à un moteur expert et une intelligence artificielle.

Le logiciel ne remplace jamais le conseiller.
Toutes les décisions finales restent humaines.

---

# Objectifs

Le logiciel devra permettre de :

- gérer un portefeuille de demandeurs d'emploi ;
- suivre les accompagnements ;
- gérer les entretiens ;
- gérer les actions ;
- gérer les formations ;
- gérer les dispositifs France Travail ;
- produire automatiquement des synthèses professionnelles ;
- produire des courriers ;
- assister le conseiller dans ses décisions.

---

# Modules prévus

Le logiciel comportera notamment les modules suivants :

- Tableau de bord
- Portefeuille
- Dossier demandeur
- Accompagnement
- Entretiens
- Actions
- Formation
- RSA
- Loi Plein emploi
- CEJ
- PMSMP
- IAE
- Handicap (RQTH)
- Employeurs
- Courriers
- Statistiques
- Administration
- Moteur Expert
- Intelligence Artificielle

---

# Principes

Le logiciel devra être :

- simple à utiliser ;
- rapide ;
- moderne ;
- sécurisé ;
- entièrement documenté ;
- évolutif ;
- modulaire.
