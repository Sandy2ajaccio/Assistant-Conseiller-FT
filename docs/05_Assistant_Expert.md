# Assistant Expert

## Mission

Assister le conseiller France Travail dans chacune de ses décisions sans jamais se substituer à lui.

---

# Objectifs

Le moteur doit :

- analyser automatiquement chaque dossier ;
- détecter les anomalies ;
- identifier les priorités ;
- proposer des actions ;
- préparer les entretiens ;
- aider à la rédaction des synthèses.

---

# Analyse automatique

Le moteur analyse :

- identité
- parcours
- inscription
- catégorie
- droits
- RSA
- Contrat d'Engagement
- RQTH
- IAE
- formation
- historique
- notes
- actions
- prescriptions
- rendez-vous
- partenaires
- freins
- compétences
- projet professionnel

---

# Détection

Le moteur détecte automatiquement :

- absence d'entretien
- absence d'action
- contrat d'engagement manquant
- dossier incomplet
- risque de rupture
- incohérences
- échéances
- anomalies administratives

---

# Recommandations

Le moteur peut proposer :

- prestation
- immersion
- PMSMP
- formation
- IAE
- orientation
- partenaire
- aide financière
- recrutement

---

# Préparation entretien

Avant chaque entretien, le moteur génère :

- résumé du dossier
- évolution
- changements
- points de vigilance
- questions proposées
- objectifs de l'entretien

---

# Après entretien

Le moteur aide à produire :

- synthèse
- plan d'action
- prochaines étapes
- actions à créer
- échéances

---

# Explication

Chaque recommandation doit être expliquée.

Le conseiller doit toujours comprendre pourquoi une proposition est faite.

---

# Traçabilité

Toutes les analyses doivent être historisées.

Aucune décision automatique.

Le conseiller reste seul décideur.

---

# Objectif final

Faire gagner du temps.

Sécuriser les décisions.

Améliorer la qualité de l'accompagnement.

Ne jamais remplacer le conseiller.
