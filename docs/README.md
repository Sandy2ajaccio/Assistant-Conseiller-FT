# Documentation - Assistant Conseiller FT

Bienvenue dans la documentation officielle du projet **Assistant Conseiller FT**.

Ce dossier contient tous les documents de référence du logiciel.

## Objectifs

Cette documentation permet de :

- définir les fonctionnalités du logiciel ;
- documenter les règles métier France Travail ;
- décrire l'architecture technique ;
- suivre les évolutions du projet ;
- faciliter le travail de développement avec GitHub Copilot et ChatGPT.

## Documents

Ce dossier accueillera progressivement :

- Cahier des charges fonctionnel
- Architecture technique
- Base de données
- Règles métier
- Roadmap
- Documentation API
- Guide développeur
- Guide utilisateur

Cette documentation est la référence officielle du projet.
