# Vision Produit
# Assistant Conseiller FT

Version : 1.0

---

# Notre ambition

Créer le meilleur assistant numérique destiné aux conseillers en insertion professionnelle.

Le logiciel ne doit pas être un simple outil de gestion.

Il doit devenir un véritable assistant métier capable de :

- faire gagner du temps ;
- sécuriser les décisions ;
- centraliser les informations ;
- réduire la charge mentale ;
- améliorer la qualité de l'accompagnement.

---

# Notre philosophie

Le conseiller reste toujours décisionnaire.

Le logiciel :

- informe ;
- analyse ;
- propose ;
- explique ;

mais ne décide jamais.

---

# Les trois piliers

## 1. Simplicité

En moins de 3 clics, le conseiller doit accéder à toute information importante.

Le logiciel doit rester simple, clair et intuitif.

---

## 2. Intelligence métier

Le moteur expert applique les règles métier.

L'intelligence artificielle reformule les analyses et aide le conseiller.

---

## 3. Gain de temps

Chaque fonctionnalité doit faire gagner du temps.

Si une fonctionnalité ne fait pas gagner de temps, elle ne sera pas développée.

---

# Les utilisateurs

Le logiciel devra pouvoir être utilisé par :

- France Travail
- Mission Locale
- Cap Emploi
- Départements (RSA)
- PLIE
- Structures IAE
- Associations d'insertion

Chaque structure pourra activer uniquement les modules qui la concernent.

---

# Notre objectif quotidien

Lorsque le conseiller ouvre le logiciel, il doit immédiatement connaître :

- ses rendez-vous ;
- ses urgences ;
- les dossiers prioritaires ;
- les actions à réaliser ;
- les alertes ;
- les échéances importantes.

---

# Notre principe

Nous ne développons jamais une fonctionnalité parce qu'elle est techniquement intéressante.

Nous la développons uniquement si elle améliore réellement le travail du conseiller.

---

# La question que nous nous poserons toujours

Cette fonctionnalité fait-elle gagner du temps au conseiller ?

Si la réponse est non, elle ne sera pas développée.

---

# Notre différence

Notre objectif n'est pas de créer un logiciel.

Notre objectif est de créer le meilleur assistant métier pour les professionnels de l'insertion.
